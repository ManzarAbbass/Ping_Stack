#PingStack – Call Reminder App

PingStack is a smart call reminder web application that helps users manage and organize call notes efficiently. It features a modern stacked card interface with priority-based categorization.

---

##Features

- Add call notes with:
  - Image URL
  - Full Name
  - Home Town
  - Purpose of call
- Assign priority categories:
  - Emergency
  - Important
  - Urgent
  - No Rush
- Stacked card UI (top 3 visible)
- Reorder cards using Up & Down controls
- Dynamic card creation using JavaScript
- Clean and modern UI design



##Tech Stack

- HTML
- CSS
- JavaScript (DOM Manipulation, Event Handling)



##How It Works

- Users create a new call note using the form.
- Each note is added as a card in the stack.
- Only the top 3 cards are visible using CSS stacking.
- Users can reorder cards dynamically using Up/Down buttons.
- New cards are added to the top of the stack.
